# — порядковий номер у поточному сортуванні. За замовчуванням — по GP Growth, тому #1 = найбільший приріст за місяць.
↑↓ — зміна позиції по GP Growth відносно останнього оновлення.
GP Now — поточний GP гравця на момент останнього снапшоту. Дані оновлюються тричі на день: 06:00, 15:00, 00:00 (Київ).
GP Growth — приріст GP з 1-го числа поточного місяця до сьогодні. На початку нового місяця скидається до нуля.
7 days — активність за останні 7 днів. ▌ зелений = GP виріс, ▌ червоний = GP не виріс, темний = немає даних.
Plan % — відсоток виконання місячного плану. План можна змінити через ⚙ у шапці.
